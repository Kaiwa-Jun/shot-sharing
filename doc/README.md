## 概要

本アプリは、カメラ初心者でも撮影設定のハードルを下げ、迷ったときにすぐ参考にできる「設定の道しるべ」を提供することを目的とした、撮影ナレッジ検索アプリです。

画像・Exif・テキストから「撮影条件→見た目」を学べる検索体験を提供し、ユーザーは自然言語や画像で質問することで、似た作例と推奨設定（ISO/F/SS/露出補正 等）を得ることができます。
