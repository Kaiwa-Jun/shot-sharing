# コンセプト

## 1. 目的 / コンセプト

### 背景（原体験）

作者自身が一眼レフカメラを趣味とし、明暗や被写体などの状況によって適切な設定を選ぶのが難しかった。その体験から、カメラ初心者でも設定のハードルを下げ、迷ったときにすぐ参考にできる**"設定の道しるべ"**を提供したい。

### コアバリュー

- 画像・Exif・テキストから「撮影条件→見た目」を学べる検索体験を提供

- ユーザーは自然言語や画像で質問し、似た作例＋推奨設定（ISO/F/SS/露出補正 等）を得る

- 撮影現場での意思決定を支援し、再現性の高い設定選びをサポート

## 2. 技術方針（RAG）

- **採用技術**: Gemini File Search（自前の埋め込み/ベクトルDBは行わない）

- **使用モデル候補**: 
  - Gemini 1.5 Flash（コスト重視）
  - Gemini 1.5 Pro（精度重視）
  - Gemini 2.0 Flash（最新・バランス型）

  ※MVPでは上記のいずれか1つに固定

- **RAG構成**: 共通RAGストアを採用（フェーズ1）。将来、ユーザー投稿をRAGへ昇格可能

- **登録単位**: 1写真 = 画像ファイル＋メタJSON（Exif/タグ/説明）

- **検索方式**: マルチモーダル（テキストのみ／画像のみ／併用）
